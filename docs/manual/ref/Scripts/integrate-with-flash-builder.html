integrate-with-flash-builder

Purpose

Adds support for Flash Builder to an existing Grails project.


   grails integrate-with-flash-builder

Description

  • modifies your .project file (but creates a backup):
    • adds the com.adobe.flexbuilder.project.flexbuilder builder
    • adds the com.adobe.flexbuilder.project.flexnature nature
    • adds the com.adobe.flexbuilder.project.actionscriptnature nature
  • creates a .flexProperties file
  • creates an .actionScriptProperties file
